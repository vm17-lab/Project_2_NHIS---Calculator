# Project_2_NHIS---Calculator
Calculator Application 🧮

A robust, interactive Graphical User Interface (GUI) calculator built with Python and Tkinter. This project demonstrates the implementation of mathematical logic, error handling, and responsive UI design for daily computational tasks.

🚀 Features

Standard Operations: Addition, subtraction, multiplication, and division.

Advanced Math: Support for exponents, square roots, and trigonometric functions.

Memory Functions: Clear (C) and All Clear (AC) functionality for efficient calculations.

Responsive GUI: A clean, user-friendly interface with organized button layouts.

Error Handling: Built-in logic to handle division by zero and invalid mathematical expressions.

🛠️ Tech Stack

Language: Python 

Library: Tkinter (Python’s built-in GUI package)

Logic: Mathematical expression evaluation using Python’s math module.

📁 Project Structure

calculator.py: The main script containing the GUI layout and calculation logic.
